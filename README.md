# GestureGuard

A real-time computer vision safety application that detects SOS hand gestures using a webcam and triggers emergency alerts.

## Features
- Webcam integration
- Real-time video processing
- SOS gesture detection (coming soon)
- Emergency notifications (coming soon)

## Tech Stack
- Python
- OpenCV
- MediaPipe
- Git/GitHub
